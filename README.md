# testapp

A React + Vite + TypeScript app using ShadCN-style components.

## Features

- Time-based greeting (morning/afternoon/evening) based on local machine time
- Name field for personalized user greeting
- ShadCN-style `Button` component with `class-variance-authority` + `tailwind-merge` + Radix `Slot`
- Tailwind CSS setup

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
